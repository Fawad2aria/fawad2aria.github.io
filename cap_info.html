Afg_Pro

Afg-Pro is an online store, where local products are available for sale.

Languages & libraries used in this Project:

  • Python
  • JavaScript
  • HTML
  • CSS
  • Django
  • Vue

Specifications

User Access

This website allowes users to login, Reset their password, and make an account. In their account, users will have access to create a new order, and to see their orders history.

Staff & Superuser Access

This website also allows admin or staff to login and access to all products, orders, and all admin needed features.
